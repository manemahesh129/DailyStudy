# Daily Study - JavaScript Learning Materials

A comprehensive collection of JavaScript learning materials covering fundamentals and core concepts.

## Repository Structure

### Part 01: Java Basics
Introduction to JavaScript and verification of the development setup.
- `01_basics.js` - Basic JavaScript concepts
- `02_step_by_step.js` - Step-by-step learning guide
- `03_verify_setup.js` - Verify your development environment
- `04_hot_code.js` - Hot code reloading examples

### Part 02: Java Concepts
Core JavaScript concepts including variables, hoisting, and functions.
- `05_comment.js` - JavaScript comments (single and multi-line)
- `06_identifier.js` - Understanding identifiers
- `07_functions.js` - Function declaration and usage
- `08_var_let_const.js` - Variable declarations (var, let, const)
- `09_lab.js` - Lab exercises on variables
- `10_hoisting.js` - Hoisting basics
- `11_hoisting_functions.js` - Function hoisting behavior
- `12_hoisting_if_block.js` - Hoisting in conditional blocks
- `13_let_if_block.js` - Let scoping in conditional blocks
- `14_var_if_block.js` - Var scoping in conditional blocks
- `15_const_if_block.js` - Const scoping in conditional blocks

### Part 03: Identifiers, Literals, Operators & Statements
Understanding identifiers, literals, operators, and their behavior.
- `16_identifier.js` - Identifier rules and conventions
- `17_literal.js` - Literal types in JavaScript
- `18_null_typeof.js` - Null type checking
- `19_null_undefined.js` - Difference between null and undefined
- `20_==and===.js` - Loose vs strict equality operators
- `21_IQ.js` - Interview questions on identifiers and literals

### Part 04: Operators
Deep dive into various JavaScript operators.
- `22_assigned_opeartor.js` - Assignment operators
- `23_increment_decrement_opeartor.js` - Increment and decrement operators
- `24_assignment_opeartor.js` - Assignment operator variations
- `25_comparision_operator.js` - Comparison operators
- `26_logical_operator.js` - Logical AND, OR, NOT operators
- `27_string_operator.js` - String concatenation operators
- `28_ternary_operator.js` - Ternary conditional operator
- `29_type_of_operator.js` - Typeof operator

### Part 05: Statements
Control flow statements including conditionals.
- `30_statements_intro.js` - Introduction to statements
- `31_conditional_if_else.js` - If-else conditional statements
- `32_if_else_if.js` - If-else-if chain statements
- `33_statuscode_real_example.js` - Real-world example with status codes
- `34_IQ_if_else.js` - Interview questions on conditionals

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/manemahesh129/DailyStudy.git
   cd CLASS\ LEARNING
   ```

2. Open any JavaScript file in your preferred code editor or IDE

3. Run the files using Node.js:
   ```bash
   node part_01_java_basics/01_basics.js
   ```

## Learning Path

Follow the files in order within each part to progress through the JavaScript concepts:
1. Start with Part 01 for setup verification
2. Move to Part 02 to learn core concepts
3. Progress through Parts 03-05 for operators and statements
4. Use the IQ files to test your understanding with interview-style questions

## Prerequisites

- Node.js installed on your system
- A code editor (VS Code, Sublime Text, etc.)
- Basic computer literacy
